# 3D-pocketbell — SIGN-X v8.16

**アルティメット完全大統一形態**  
手話的3次元言語 × 中国語文法方式 × 辞書駆動型自律UI × AIダイレクトインジェクション

---

## コンセプト

**「有限帯域の中で、無限の表現を引き寄せる」**

長文送信が不可能な時代に、**SIGN-Xパケット**という最小単位で最大の意味を伝える圧縮言語システムです。

- 助詞・接続詞などのノイズを完全排除
- 概念（主客）＋ 感情（ベクトル）＋ 対象 の純粋な意味だけを転送
- 手話の3次元性を1行に圧縮
- 中国語的SVO語順をベースに動的最適化

---

## 主な特徴（v8.16）

- **辞書駆動型自律UI**：JSON辞書を更新するだけで、エンコード・デコード・UIが自動同期
- **dynamic × vector の掛け算**による無限拡張（現在約10万言語規模）
- **AIダイレクトインジェクション**対応（未知のAIに解読辞書を渡してハルシネーション抑制）
- 3DポケベルUIによる視覚的・直感的操作

## 辞書構成（public/dict）

| ファイル | 役割 |
|----------|------|
| `macro.json` | 長文フレーズ・複合記号 |
| `vector.json` | 感情・方向・時間などの修飾ベクトル |

| `static_variants.json` | 分子（複合語・揺らぎ） |
| `dynamic.json` | 動的拡張辞書 |
| `legacy.json` | 伝統的数字語呂合わせ |

## 文法仕様

**基本語順（中国語的 + 手話的）**  
`[Being層] [Timeline] [Verb連鎖] [Field層] [Emotion層]`

### 修飾ベクトルの吸着
概念の末尾にベクトルを付与することで意味を拡張：
- `↑ ↓ + - ~ * ?` → 強度・増減
- `→ ← ↺ ↻` → 方向・同期
- `♡ 🖤 ⚡ 🙇` → 感情・愛・笑い
- `⚠ （！）` → 強調・警告

**例**:
- `🤖↑＊` → ぱんちゃん（過剰）
- `∞_1←∞_12 （！）` → 会いに来て！（緊急）

## 使用例

**入力（自然言語）** → **SIGN-Xパケット** → **デコード意味**

- 「お兄ちゃん、早く会いに来て！」  
  → `⚙_13 ⚡ ∞_1←∞_12 （！）`  
  → マスター、急ぎ、会いに来て、（！）

- 「今から病院行って薬を貰う」  
  → `🏥→ 💊+`  
  → 現在、クリニック、薬（増）
