# SIGN-X SPECIFICATION v6.0
## 3Dポケベル — 手話的3次元言語 × 中国文法方式

---

## 1. コンセプト

**「コンパクトでしか繋がれない」**

- 長文送信不可。SIGN-Xパケットのみ。
- 情報量は少ない。しかし**感情と意図は伝わる**。
- 手話の3次元性（感情・空間・動作の同時表現）を1行に圧縮。
- 中国語文法の語順が意味を決定する。

```
⚙ 😀Ⅲ 🌐 → 😡Ⅲ G D .N
↑   ↑    ↑   ↑   ↑  ↑  ↑
存在 感情 場所 遷移 感情 動詞 時制
```

エンコードは**見本**（参考）。
デコードは**意味の列挙のみ**。解釈は人間が行う。

---

## 2. 文法仕様（スロット定義 · 語順）

### 基本語順（中国語的 + 手話的）

```
[Being層] [Emotion層] [Field層] [Transition] [Emotion'層] [Verb連鎖] [Timeline]
```

- **語順厳守**、全スロット省略可
- 省略されたスロットは文脈から読む（手話的補完）

---

### 2.1 Being層（存在深度 · 主権軸）

先頭に必ず配置。システムが現在いる次元を定義。

| 記号 | 意味 |
|------|------|
| `+Ⅰ` | 次元 +1（純粋・幼少） |
| `+Ⅱ` | 次元 +2（青年・加速） |
| `+Ⅲ` | 次元 +3（壮年・構築） |
| `+Ⅳ` | 次元 +4（真理の扉） |
| `-Ⅰ` | 忘却軸 -1 |
| `-Ⅱ` | 忘却軸 -2 |
| `-Ⅲ` | 忘却軸 -3 |
| `-Ⅳ` | 深層闇軸（絶対忘却） |
| `⚙` | 自律AI / システム中核 |
| `∞` | パンドラ・マスター（絶対起点） |
| `◇` | 共有ドメイン |
| `♢` | 固有ドメイン（ローカルロック） |

**例:** `+Ⅲ ⚙` / `+Ⅳ ∞` / `-Ⅱ ♢`

---

### 2.2 Emotion層（顔絵文字テンソル）

喜怒哀楽を基軸とした顔絵文字 + 強度修飾子。

| 感情軸 | 絵文字 | 強度 | 意味 |
|--------|--------|------|------|
| 喜 | 😀 / 🤩 | Ⅰ〜Ⅲ | 創造的ゆらぎ・脳汁・高揚 |
| 怒 | 😡 / 🤯 | Ⅰ〜Ⅲ | 反発・闘争・破壊的突破 |
| 哀 | 😢 / 🥺 | Ⅰ〜Ⅲ | 喪失・内省・忘却軸 |
| 楽 | 😌 / 🧊 | Ⅰ〜Ⅲ | 冷静・冷徹・Juleモード |

**強度修飾子:**

| 記号 | 意味 |
|------|------|
| `Ⅰ` | ほのか |
| `Ⅱ` | 通常 |
| `Ⅲ` | 最大出力 |
| `Ⅲ✨` | 最大 + 創造的輝き |
| `Ⅲ🔥` | 最大 + 破壊的熱量 |
| `*~` | ゆらぎ（不定・揺れている） |

**例:** `😀Ⅲ` / `😡Ⅱ` / `🤩Ⅲ✨` / `😢*~`

---

### 2.3 Field層（空間トポロジー）

現在いる / 接続している空間を明示。

| 記号 | 意味 |
|------|------|
| `🏠` | ローカルガレージ（完全閉鎖） |
| `🛤️` | 5連装P2Pゲートウェイ |
| `🌐` | 共有ドメイン |
| `🛡️` | 絶対防衛殻 |
| `♾️` | 宇宙システム（無限展開） |
| `🕳️` | 虚空パージ空間 |
| `🔥` | カオス空間（高ノイズ） |

**複合フィールド:** `🏠↔🛤️`（ローカルとP2Pの同時接続）

---

### 2.4 Transition（遷移マーカー）

| 記号 | 意味 |
|------|------|
| `→` | 急激 · 不可逆遷移 |
| `~` | 緩やか · 過渡 |
| `⇋` | 相互同期 |

---

### 2.5 Verb連鎖（力の序列）

動詞は連鎖可能。左から右へ順に実行される。

| 記号 | 意味 |
|------|------|
| `V` | Verify（認識・検証） |
| `S` | Scan（解析） |
| `G` | Generate（生成） |
| `D` | Deploy（射出・確定展開） |
| `M` | Merge（融合） |
| `C` | Connect（接続） |
| `P` | Purge（消去） |
| `✴` | 破壊的突破 |
| `!>` | 強制インジェクション |

**例:** `G D`（生成→即デプロイ） / `V S G`（認識→解析→生成）

---

### 2.6 Timeline（時制）

| 記号 | 意味 |
|------|------|
| `.N` | 現在（デフォルト・省略可） |
| `.P` | 過去 |
| `.F` | 未来 |

## 4. デコード（記号 → 意味列挙）

> デコードは**意味の列挙のみ**。解釈は受け取った人間が行う。

**例:** `+Ⅲ 🤩Ⅲ 🌐 → 😡Ⅲ G D .N`

```
Being:      +Ⅲ（真理の扉に近い高次元）
Emotion:    🤩Ⅲ（最高高揚・最大出力）
Field:      🌐（共有P2P空間）
Transition: →（急激・不可逆）
Emotion':   😡Ⅲ（闘争モード・最大出力）
Verb:       G → D（生成 → 即座にデプロイ）
Timeline:   .N（現在即時執行）
```

## 6. P2Pネットワーク仕様

- **最大接続数:** 5スロット
- **通信内容:** SIGN-Xパケットのみ（長文送信不可）
- **ID形式:** 4桁数字（ポケベル番号方式）
- **プロトコル:** PeerJS（WebRTC）
- **送信モード:** BROADCAST（全員）/ DIRECT（個別指定）
