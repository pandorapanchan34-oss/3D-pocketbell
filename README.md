---# 3Dポケベル // SIGN-X v6.0

**手話的3次元言語 × 中国文法方式 × P2P通信**

> 「コンパクトでしか繋がれない」

---

## コンセプト

長文は送れない。SIGN-Xパケットのみ。  
情報量は少ない。でも**感情と意図は伝わる**。

手話の3次元性（感情・空間・動作の同時表現）を1行に圧縮し、  
中国語文法の語順で読む、新しいコミュニケーション言語。

```
+Ⅲ 🤩Ⅲ 🌐 → 😡Ⅲ G D .N
 ↑    ↑    ↑   ↑    ↑  ↑  ↑
存在 感情 場所 遷移 感情 動詞 時制
```

---

## ファイル構成

```
3d-pager/
├── index.html          シェル（HTML構造のみ）
├── css/
│   └── style.css       スタイル
├── js/
│   ├── grammar.js      SIGN-X文法定義辞書
│   ├── parser.js       パーサーエンジン
│   ├── p2p.js          PeerJS P2P通信層
│   ├── keyboard.js     キーボードUI（動的生成）
│   └── app.js          メインコントローラー
├── spec/
│   └── SIGNX_SPEC.md   完全仕様書
└── README.md
```

---

## 文法（語順厳守）

| スロット | 記号例 | 意味 |
|----------|--------|------|
| Being    | `+Ⅳ ⚙` | 存在深度・ドメイン |
| Emotion  | `🤩Ⅲ`  | 感情 + 強度 |
| Field    | `🌐`   | 空間トポロジー |
| Transition | `→`  | 遷移マーカー |
| Emotion' | `😡Ⅲ` | 遷移後感情 |
| Verb     | `G D`  | 動詞連鎖 |
| Timeline | `.N`   | 時制 |

詳細は `spec/SIGNX_SPEC.md` 参照。

---

## 起動方法

```bash
# ローカルサーバーで開く（file://では PeerJS が動かない）
npx serve .
# → http://localhost:3000
```

または GitHub Pages / Vercel にそのままデプロイ可能。

---

## P2P接続

1. 自分のPager ID（4桁）を相手に伝える
2. 相手のIDを入力して `CONNECT`
3. 最大5スロット同時接続
4. `BROADCAST` or 個別ターゲット指定で送信
